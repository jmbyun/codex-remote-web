Remote Codex on iPhone

Control OpenAI Codex CLI on your SSH server from anywhere.

Codex Mobile IDE is a mobile client for developers who run OpenAI Codex CLI on a remote server. Connect over SSH, browse files, open a shell, resume Codex sessions, and keep working from iPhone or iPad.

Get Support Requirements
SSH-first workflow Your code stays on the remote machine, not on the device.
Codex session history Resume previous Codex runs directly from the server’s session files.
Online only This app requires internet access and a reachable SSH server with Codex CLI installed.
Files, shell, and Codex sessions in one flat mobile workspace.

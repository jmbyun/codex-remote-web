Privacy Policy

Last updated: March 1, 2026

This Privacy Policy explains how Codex Mobile IDE handles information when you use the app. Codex Mobile IDE is a mobile client for connecting to your own remote server over SSH and interacting with OpenAI Codex CLI running on that server.

What information the app uses

  • Server connection details you enter, such as server name, host, port, username, and optional password.
  • Remote file paths, shell commands, and Codex session content needed to display and continue your remote workflow.
  • Locally stored preferences such as saved servers and the last opened remote directory for each server.

How information is used

  • To establish an SSH connection to the server you choose.
  • To browse remote files, open a remote shell, and start or resume Codex sessions.
  • To remember app state such as saved server entries and recently opened directories.

Where your data is processed

Codex Mobile IDE is a remote access client. Commands, file operations, and Codex interactions are performed on the remote server you connect to. The app itself does not provide an offline coding model and does not work without a reachable server and active internet connection.

Data storage

  • Saved server information may be stored locally on your device.
  • Remote shell output, file contents, and Codex session history are displayed from your remote server and may remain on that server according to its own configuration.
  • The app does not claim ownership of your repositories, remote files, or Codex session logs.

Third-party services

The app connects to systems you configure, including your SSH server and any services used by OpenAI Codex CLI on that server. Your use of those services is governed by their own terms and privacy policies.

Data sharing

Codex Mobile IDE does not sell your personal information. The app only transmits information needed to connect to your server and operate the remote workflow you request.

Security

The app uses SSH to communicate with your remote server. However, no method of transmission or local storage is completely secure, and you are responsible for the security of the server, credentials, and Codex environment you choose to use.

Children's privacy

Codex Mobile IDE is intended for developers and is not directed to children.

Changes to this policy

This Privacy Policy may be updated from time to time. The latest version will be posted on this page.
